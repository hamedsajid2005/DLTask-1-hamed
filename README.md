# To-Do List Application

## Description

This is a simple command-line To-Do List application developed using Python. The program allows users to add tasks, view their task list, check the total number of tasks, and exit the application. It demonstrates the use of Python lists, loops, conditional statements, and user input handling.

## Features

* Add new tasks
* View all tasks
* Show total number of tasks
* Simple menu-driven interface

## Technologies Used

* Python 3

## How to Run

1. Make sure Python is installed on your system.
2. Download or clone the project files.
3. Open a terminal or command prompt in the project folder.
4. Run the following command:

```bash
python todo_list.py
```

## Example Usage

1. Select "Add Task" to enter a new task.
2. Select "View Tasks" to display all saved tasks.
3. Select "Show Total Tasks" to see the number of tasks.
4. Select "Exit" to close the application.
